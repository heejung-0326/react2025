02_자바스크립트 함수 - map

JavaScript의 map() 함수는 배열의 각 요소에 대해 주어진 함수를 호출하고, 그 결과를 모아서 새로운 배열을 반환합니다. 원본 배열은 변경되지 않습니다.

map함수는 원본 배열은 변경하지 않습니다.

배열의 각 요소에 함수를 적용하여 새로운 배열을 반환합니다.

배열데이터를 불러와 주로 변환이나, 속성을 추출하거나 계산 등의 작업시 유용

filter, forEach와 같은 구문

    문법1
    array.map(function(currentValue, index, array){
      //새로운 배열
    })

    문법2
    array.map(function(currentValue, index, array),thisArg{
      //새로운 배열
    })
  • currentValue 현재 처리 중인 요소
  • index(선택) : 현재 요소의 인덱스
  • array(선택) : map을 호출한 배열
  • thisArg : callbackFunction내에서 this로 사용될 값

1. for문을 활용하여 숫자데이터 출력하기

2. map함수를 사용하여 숫자 데이터 출력하기

3. 요일 데이터 앞에 특수문자, 이모지 아이콘 추가하기

4. 배열 객체에서 영문자 소문자를 대문자로 변경하기

5. 배열 객체에서 특정 속성만 추출하기

6. 사용자 배열 객체에서 특정 조건에 맞는 사용자들의 이름만 뽑아서 문자열로 합치기

  • filter() 함수 - 조건을 만족하는 요소만 추출하여 새로운 배열을 반환합니다.
    문법) array.filter(callback(currentValue, index, array), thisArg)
  • reduce() 함수 - 배열을 **하나의 값으로 축약(누적)**합니다. 예: 합계, 곱, 객체 합성 등
    문법) array.reduce(cllback(accumulator, currentValue, index,array), initialValue)
